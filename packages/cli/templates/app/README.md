Getting Started

Follow these steps to start developing your application.

1. Install dependencies

If your environment requires dependencies, install them first.

2. Start the development server

Run the development server:

najumi dev

This will start the Najumi development server.

Open your browser and navigate to:

http://localhost:3000

You should see your application running.

---

Project Structure

The project uses the standard Najumi application structure.

src
 ├ pages        → Frontend pages
 ├ api          → Backend API routes
 ├ components   → Reusable UI components
 └ layouts      → Page layouts

modules         → Najumi modules

public          → Static files

najumi.config.js → Application configuration

Development

The development server supports features designed to improve developer productivity.

These include:

- automatic file watching
- fast reload during development
- integrated backend and frontend execution

Start development:

najumi dev

---

Building for Production

To create an optimized production build, run:

najumi build

This command prepares your application for production deployment.

---

Running in Production

After building the project, start the production server:

najumi start

This will launch the optimized production runtime.

---

Environment Configuration

Environment variables can be configured using the ".env" file.

Example:

cp .env.example .env

Edit the ".env" file to configure your application settings such as:

- database connection
- authentication secrets
- external API keys
